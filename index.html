pebble

This entire website was generated by pebble — a no-nonsense, minimal, lightweight website generator written in Python.

No frameworks. No JavaScript. No build pipeline. Just Markdown files converted to plain HTML and served fast.

Write a .md file, run ./build.sh, done.

Why?

Because every other tool wants you to learn its config DSL, install 400 npm packages, and wait 30 seconds for a hot-reload. You just want a website.

Write, build, done. Your nav builds itself, your RSS feed writes itself, and nothing else happens that you didn't ask for.

Get it

git clone https://github.com/erdemuysalx/pebble
cd pebble
uv sync

Edit config.yml with your details, put your posts in blog/, run ./build.sh. Output lands in build/. Deploy anywhere.
